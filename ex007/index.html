principais formatações de texto

tags come sem semantica

tags sem semantica nao possuem significado para o html, pois são tags de forma. Sendo mais antigas, nao é recomendado usar na parte de html, pois podem ficar obsoletas, já que passaram ou passarão para a parte de css. Já as tags com semantica possuem significado para o html, e são mais recomendadas para uso no html

negrito/destaque

nesta frase tenos um termo em negrito usando a tag b (termo não semantico)

nesta frase, tenho um termo em negrito usando a tag strong (termo semantico)

italico / enfase

nesta frase, temos um termo em italico usando a tag i (nao semantica)

nesta frase, temos um termo em italico utiizando a tag em (semantica)

texto marcado

podemos criar tambem textos marcados usando a tag mark

e no outro paragrafo, temos um termo marcado usando a mesma tag

para manter uma cor em todos os marks, adicione um style no cabeçalho da pagina html e mude a cor padrão

texto grande e pequeno

temos um texto grande usando a tag big (tag obsoleta) e um texto pequeno usando a tag small

texto deletado

ppodemos marcarum texto como texto excluido para inicar que ele deve ser lido, mas não considerado, usando a tag del

texto inserido

podemos marcar um texto como texto inserido para indicar que ele foi adicionado depois, usando a tag ins

texto sobreescrito

serve para inserir coisas do tipo x20+3, se usa a tag sup

texto subescrito

para inseiri coisas do tipo H2O, se usa a tag sub
